# Tipos de variables:
Locales: se declaran dentro de una funcion y no se guardan en la cadena.
Estado : Se declaran fuera de las funciones y se guardan en la cadena.
Globales: Son las variables que nos proveen de informacion de la cadena.
# Visibilidad
- external: llamados por otros contratos.
- public: pueden ser llamados por el contrato y por otros, automaticamente se crea una funcion get.
- internal: pueden ser llamados solo donde se definieron dentro de un contrato y contrados derivados.
- private : puede ser llamados solo en el contrato que se definieron.
